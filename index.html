Things I've built.

Automation · Customer Support
BobBot — CS Automation System for Bobplus Africa
Built from experience working as a CS agent at Bobplus Africa — every feature addresses a real pain point. BobBot handles merchant queries on Microsoft Teams, while a background transaction monitor checks every 5 minutes and escalates based on urgency: CS reminder at 12hrs, supervisor at 24hrs, instant email at 48hrs, pattern alert for 10+ failures. CS agents work from one real-time dashboard — merchant chat, alerts feed, and full transaction history, all without switching apps. Runs in demo mode without an API key.

Mobile · Final Year Project
AttendX v2 — Smart Attendance System
A Flutter mobile app that automates student attendance at JKUAT using GPS geofencing and Bluetooth beacon detection. Three user roles — students, teachers, and admins — each with tailored dashboards. Anti-proxy measures include device binding and biometric authentication. Cloud Functions auto-close sessions and push at-risk alerts when attendance drops below 75%.

ML · FinTech · Group Project
Credit Scoring via Mobile Money Behaviour
A 7-phase ML pipeline simulating 1.96 million synthetic M-Pesa transactions for 5,000 users. Compares a rule-based deterministic baseline against a Random Forest classifier — the AI model achieved AUC-ROC of 0.9958 and rescued 97 creditworthy borrowers from wrongful loan rejection. Includes SHAP explainability and a Streamlit dashboard with a live What-If simulator. Supervised by Dr. Phineas Roy Kiogora, JKUAT.

ML · Learning Project
Titanic Survival Prediction
A supervised ML project covering the full workflow — EDA, data cleaning, feature engineering, and model evaluation. Trained and compared Decision Tree, Naive Bayes, and Random Forest classifiers on 891 passengers. Random Forest achieved 82.1% accuracy. Key finding: sex (28%) and fare (27%) were the strongest predictors of survival.
